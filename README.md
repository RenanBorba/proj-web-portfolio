# Projeto Portfólio - Sushi Bar Web
Projeto Front-End de web site para Portfólio de Sushi Bar e bares em geral, em Bootstrap 4. 
<ul>
  <li>Npm</li>
  <li>Sass</li>
  <li>Fonts</li>
  <li>Font Awesome</li>
  <li>Magnific-Popup</li>
  <li>Slick</li>
  <li>Google Maps IFrame</li>
</ul>
